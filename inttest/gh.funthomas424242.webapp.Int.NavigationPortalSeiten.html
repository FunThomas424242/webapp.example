Narrative:

In order to die Anwendung vollständig benutzen zu können
As a Koch
I want to zwischen den einzelnen Seiten navigieren können Damit auch an die richtigen Stellen in der Anwendung gelangen kann.

Scenario: Navigation vom Default zur Welcome Seite

Gegeben sei die Webserver Übersichtsseite.
Wenn der erste Link geklickt wird
Dann wird zur Welcome Seite navigiert.

Scenario: Navigation zur Login Seite

Gegeben sei als aktuelle Seite die Welcome Seite.
Wenn der Link Anmelden geklickt wird
Dann wird zur Login Seite navigiert.
